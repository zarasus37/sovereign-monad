## APIs Requiring Keys

### Alchemy (Recommended)
1. Go to https://www.alchemy.com/
2. Sign up for free account
3. Create new app (Base, Arbitrum, Monad testnets)
4. Copy API key

**Free tier:** 3 apps, 100M compute units/month

### Infura (Alternative)
1. Go to https://www.infura.io/
2. Sign up for free account
3. Create new API key
4. Enable desired networks

**Free tier:** 100K requests/day

### Twitter/X API
1. Go to https://developer.twitter.com/
2. Apply for developer account (takes ~24-48 hours)
3. Create app to get API keys
4. Copy API Key and API Secret

**Free tier:** 1.5K tweets/month (read-only)
