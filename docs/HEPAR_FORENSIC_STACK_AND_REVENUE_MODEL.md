# Hepar Forensic Stack and Revenue Model

Date context: 2026-04-29

This document defines the upgraded Hepar operating model and how six-organ outputs convert into commercial products without changing the organism's core runtime mandate.

## 1) Hepar Forensic Stack (v2)

Hepar now evaluates opportunities through five explicit forensic layers:

1. Bytecode Privilege Checks
2. Proxy-Admin Control Analysis
3. LP Unlock Concentration and Timeline
4. Wallet Graph Taint and Behavioral Patterning
5. Adversarial Execution Simulation

### Layer outputs

Hepar returns:

1. hard-block reasons
2. weighted risk vector
3. aggregate risk
4. confidence level (`low`, `medium`, `high`)
5. detailed reasons for operator review and downstream narrative packaging

### Hard-block examples

1. unchecked owner-drain functions
2. selfdestruct reachable
3. uninitialized proxy implementation
4. single-actor upgrade path with timelock bypass
5. concentrated near-term LP unlock cliff
6. creator wallet taint + repeated rug pattern
7. adversarial invariant break with material exploit estimate

## 2) Six-Organ Productization Logic

Commercial value is treated as packaging of existing organ outputs:

1. `Synapse` routes signal and urgency.
2. `Hepar` produces forensic risk truth.
3. `Cortex` synthesizes explainable institutional intelligence.
4. `Vox` packages into report/feed/public forms.
5. `Pneuma` converts outbound distribution and lead flow.
6. `Cardia` preserves treasury rhythm and reinvestment discipline.

No additional doctrine claim is needed. The revenue surface is already implicit in current runtime behavior.

## 3) Revenue Tiers (Optimized Packaging)

### Tier 1: Hepar Risk API

- product: continuous protocol surveillance
- pricing shape: `$0.01/check` usage lane or `$50k/mo` unlimited commit lane
- primary organ producers: `Hepar`, `Synapse`, `Vox`, `Pneuma`
- key KPI gates: daily protocol checks, false-positive rate, critical-detection precision

### Tier 2: Institutional Risk Feed

- product: live risk intelligence feed for funds and market makers
- pricing shape: `$50k-$500k/mo` based on latency, coverage, and support tier
- primary organ producers: `Hepar`, `Cortex`, `Vox`, `Pneuma`
- key KPI gates: institutional conversion, retention/churn, signal hit-rate vs realized incidents

### Tier 3: Protocol Forensic Report

- product: deep pre-mainnet forensic workup
- pricing shape: `$5k` basic report / `$25k` institutional deep-dive + remediation retest upsell
- primary organ producers: `Hepar`, `Cortex`, `Vox`
- key KPI gates: turnaround time, close rate per outbound lead, retest upsell conversion

Tier 3 fast-lane operational polish (target `95+` readiness):

1. automated report templating
2. quality checklist + sign-off workflow
3. automated evidence packaging with proof-link export
4. customer portal intake with 24-hour target turnaround
5. explicit SLA guarantees for response-time and quality commitment

### Tier 4: Insurance Underwriting Dataset

- product: licensing-quality protocol risk dataset
- pricing shape: annual licensing first (`$10M-$50M`) with optional premium-share lane (`2-5%`) after actuarial trust is proven
- primary organ producers: `Hepar`, `Cortex`, `Synapse`, `Cardia`, `Pneuma`
- key KPI gates: partner underwriter count, refresh SLA, insured loss-ratio improvement vs baseline
