# Delivery Process

## When Customer Buys

### Step 1: Confirm Payment
- Check Stripe/payment notification
- Note customer email & name

### Step 2: Prepare Delivery
- Copy template to new folder
- Remove any private info
- Add customer name to README

### Step 3: Send Delivery Email
```
Subject: Your [Product] is Ready!

Hi [Name],

Thank you for your purchase!

[Product] is ready for you.

[Download Link / GitHub Link]

Let's schedule your free setup call:
[Calendly Link]

Questions? Reply to this email.

Best,
[Your Name]
```

### Step 4: Schedule Setup Call
- Send Calendly link
- 30-minute call
- Help them get started

---

## Customer Checklist

Before delivery, make sure:
- [ ] Payment confirmed
- [ ] Template tested
- [ ] Documentation updated
- [ ] Setup call scheduled
- [ ] Delivery email sent

---

## Refund Process

If customer requests refund:
1. Verify payment
2. Process refund in Stripe
3. Send confirmation email
4. Archive their info
