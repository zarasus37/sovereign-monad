# emergence-observer-core

Local emergence observation preparation for Sovereign Monad.

## What it does

- evaluates whether the ecosystem has enough differentiated roles, feedback, continuity, and mutual contribution to justify longitudinal observation
- keeps the output observational only
- does not claim emergence and does not activate any economic surface

This package is a readiness and evidence surface. It is not an emergence declaration engine.

## Run

```powershell
cd emergence-observer-core
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```
