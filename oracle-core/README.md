# oracle-core

`oracle-core` is the local analysis-mode Oracle scaffold for the Sovereign Monad ecosystem.

It takes verified local signal posture and converts it into:

- regime classification
- deployment posture
- commercialization posture
- explicit reasons for the current recommendation

It does not claim live calibration or production buyer proof. It exists so Layer 6 is no longer only repo-adjacent logic.

## Commands

```bash
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```

## Runtime truth

This package is local and analysis-only.

It does not:

- price real markets
- replace the risk engine
- claim production Oracle maturity

It gives Layer 6 a concrete executable scaffold that can be tightened once live data and buyer proof exist.
