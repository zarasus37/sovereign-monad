# population-growth-core

Local population growth planner for the Data Rail.

## What it does

- measures the current behavioral sample against the defined diversity thresholds
- identifies the exact population gaps blocking externalization
- produces a bounded growth plan for actors, classes, surfaces, outcomes, and concentration

This package does not manufacture diversity. It only makes the missing population shape explicit.

## Run

```powershell
cd population-growth-core
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```
