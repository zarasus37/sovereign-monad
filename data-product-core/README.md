# data-product-core

Local external Data Rail productization preparation scaffold.

This package prepares product bundles and externalization scope without claiming live external sale, live routing, or active externalization.

`lightverify-core` sits beside this package as the commercial certification/provenance surface for bundles that clear integrity, rights, and de-identification review. `data-product-core` prepares the bundle. `lightverify-core` does not authorize runtime execution and does not replace externalization gating.

Current bundle examples now include a six-organ revenue packaging surface:

- Tier 1: Hepar continuous risk API
- Tier 2: institutional risk feed
- Tier 3: protocol forensic pre-mainnet reports
- Tier 4: insurance underwriting risk dataset

These are preparation surfaces only and remain bounded by governance, rights review, activation scope, and externalization gates.
