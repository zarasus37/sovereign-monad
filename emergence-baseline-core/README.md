# emergence-baseline-core

Local longitudinal baseline surface for emergence observation.

## What it does

- compares multiple observation windows over time
- tracks readiness drift and continuity across windows
- prepares a seed longitudinal baseline without claiming emergence

This package remains observational only.

## Run

```powershell
cd emergence-baseline-core
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```
