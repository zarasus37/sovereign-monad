# signal-layer

`signal-layer` is the zero-capital signal substrate for the Sovereign Monad ecosystem.

It provides:

- signal schema normalization
- cross-domain aggregation
- initial behavioral and operational interpretation
- a local analysis-mode runtime snapshot

It does not claim live production ingestion. It gives Layer 7 an executable local surface so signal handling is no longer only doctrine.

## What it covers

- signal schema
- aggregation substrate
- behavioral and event interpretation layer

## Runtime truth

This package is analysis-driven.

It does not:

- consume live chain events
- claim institutional calibration
- replace Dove, Oracle, or Gnosis

## Commands

```bash
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```

## Runtime output

`npm start` prints:

- normalized sample signal count
- aggregate counts by domain, severity, source, and lane
- interpretation labels for:
  - coordination pressure
  - boundary tension
  - exchange readiness
  - capital attention
  - operator load

That gives Layer 7 a runnable local base before live ingestion exists.
