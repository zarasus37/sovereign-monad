# expansion-control

Controlled expansion framework for the local Sovereign Monad platform layer.

## What it does

- evaluates proposed local or outward expansion requests
- blocks capital-facing or outward-facing work when posture is not healthy enough
- keeps internal local growth available even under stressed posture when it does not violate boundaries

This package is local-analysis-only and policy-driven.

## Run

```powershell
cd expansion-control
cmd /c npm install
cmd /c npm run build
cmd /c npm test
cmd /c npm start
```
